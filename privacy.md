1. Introduction
This Privacy Policy outlines how typicaldcbot ("the Bot") handles data. We are committed to protecting user privacy and only collecting data necessary for the Bot's functionality.

2. Data We Collect
The Bot may collect the following data points:

User IDs: To identify users for features like leveling, moderation, and user-specific settings.

Server IDs: To store server-wide configurations (e.g., custom prefixes, log channels).

Message Metadata: We process message IDs and timestamps to facilitate moderation logs and command responses.

User-Provided Content: Any content explicitly provided to the bot through commands (e.g., custom tags or bios). We do not log or store general message content outside of these interactions.

3. Use of Data
Data is used strictly to provide the Bot's services. We do not sell, rent, or share user data with any third parties.

4. Data Retention and Deletion

Retention: Data is stored as long as the Bot is in your server or until a deletion request is made.

Deletion: Users may request the removal of their data at any time by joining our support server at https://discord.gg/3KSENXccUn
